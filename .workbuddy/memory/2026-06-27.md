## 项目概览阅读
- 阅读了 MacRelay 项目的 README、Package.swift、需求文档（docs/AI 编程 CLI 客户端需求文档.md）
- 项目是一个 local-first 的 macOS ↔ iPhone 中继工具，让 iPhone 通过局域网远程操控 Mac 上的 AI coding CLI（Codex CLI、Claude Code 等）
- 技术栈：Swift/SwiftUI 原生双端，Swift Package Manager，HTTP + WebSocket 通信协议
- 模块分层：AgentClientCore（核心协议/状态机/认证）→ AgentClientIO（HTTP/WS 客户端）→ 应用层（Mac Shell + iOS App）
- 当前进度：配对、中继、状态同步主干已实现；App Store 分发、完整 approval live、加密 QR 等尚未完成
- docs 目录有完整需求文档、协议文档、UI设计基准、落地执行计划
